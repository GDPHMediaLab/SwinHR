# SwinHR
For the paper "SwinHR: Hemodynamic-Powered Hierarchical Vision Transformer for Breast Tumor Segmentation". We also openly publish the tumor annotation of the public dataset used in this paper. If you need them for your research, please contact [Dr Zhenwei Shi](shizhenwei@gdph.org.cn), we are happy to share.
